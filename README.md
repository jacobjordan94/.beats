A live [Swatch Internet Time](https://en.wikipedia.org/wiki/Swatch_Internet_Time) clock. The day is divided into **1000 .beats** — no time zones, one universal time for the entire internet.

**[dot-beats.jacob-jordan.me](https://beats.jacob-jordan.me)**

## Features

- Real-time .beat display with animated slot digits
- Expandable decimal precision
- Day progress bar with percentage
- Light / dark theme

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

React 19, TypeScript, Vite, Tailwind CSS, Motion, shadcn/ui
